# Claude Replica

A full-featured replica of the Claude chatbot interface, built with React and Vite.

## Prerequisites

- Node.js (v18 or higher)

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Open http://localhost:5173 in your browser.

## Features

- **Multi-LLM Support**: OpenAI, Groq, Gemini.
- **Artifacts**: Dedicated side panel for code and content previews.
- **Web Search**: Integrated Tavily search.
- **Secure**: API keys are stored locally in your browser.
